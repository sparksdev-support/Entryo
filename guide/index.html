Entryo is a simple personal accounting app that supports multiple ledgers, income and expense entries, category statistics, budgets, mood records, location records, import/export, and iCloud sync.

This guide explains the main feature locations and common workflows so you can start using Entryo quickly.

1. Quick Start

Create a Ledger

When you open Entryo for the first time, you can create a new ledger. Ledgers help you separate different accounting scenarios, such as:

  • Daily life
  • Travel
  • Pets
  • Family expenses
  • Side income

When creating a ledger, you can set its name, default currency, ledger color, and other details.

The default currency is used for statistics, budgets, and multi-currency conversion. After creating a ledger, you can still adjust some details in Ledger Settings.
To open Ledger Settings, go to the ledger list, swipe left on the ledger, then tap “Edit”.

Record Your First Entry

After entering a ledger, tap the add button to record an entry.

You can choose:

  • Income or expense
  • Amount
  • Category
  • Date
  • Note
  • Mood
  • Location
  • Whether to exclude the entry from daily statistics

You can complete an entry with only an amount and category. Other fields can be added when needed.

Switch Ledgers

If you have created multiple ledgers, you can switch between them in the ledger list. Each ledger has its own entries, categories, budgets, and settings.

Free Version and Pro

Entryo’s basic features can be used for free. Pro is a one-time purchase, not a subscription.
You can continue using the basic accounting features for free, or upgrade to Pro to unlock more ledgers, more subscriptions, custom appearance, mood statistics, location statistics, category budgets, and more.

For details about Pro features, purchase, and restoring purchases, see Pro and Restore Purchase.

2. Entries

Add an Entry

On the ledger page, tap the add button to create an income or expense entry.

An entry can include the following fields:

FieldDescription
TypeIncome or expense
AmountThe amount of this entry
CategoryFor example, food, transportation, salary, etc.
DateWhen the entry occurred
NoteAdditional details, merchant name, purpose, etc.
CurrencyUsed for entries in a non-default currency
MoodHow you felt when the income or expense occurred
LocationWhere the entry occurred
Exclude from daily statsWhen enabled, this entry is excluded from related statistics but remains in the entry list

Edit an Entry

In the entry list, swipe left on an entry, then tap “Edit” to open the entry details and modify it.

You can modify the amount, category, date, note, currency, mood, location, and whether the entry is excluded from daily statistics.

After editing, related statistics will be recalculated based on the updated data.

Delete an Entry

In the entry list, swipe left on an entry, then tap “Delete” to delete it.

After deleting, an undo prompt appears at the bottom of the page. If the deletion was a mistake, tap “Undo” immediately to restore the entry.

If the undo prompt has disappeared, you can also go to “Top-right Menu” → “Recently Deleted” to restore or permanently delete entries.

Note: Recently Deleted only keeps recently deleted records on the current device. It does not sync via iCloud and is not included in ledger backups.

Exclude from daily stats

If you do not want an entry to affect daily statistics, enable “Exclude from daily stats”.

This is useful for:

  • Expenses that will be reimbursed later
  • Account adjustments
  • Temporary records
  • Special entries you do not want to include in daily spending analysis

When enabled, the entry remains in the list but is excluded from related statistics. The exact scope follows what is shown in the app.

You can set this option when adding or editing an entry. When enabled, the entry remains in the entry list but is excluded from related statistics.

On the statistics page, you can also use a switch to show these entries when you need to view complete records.

Quick Entry Bar

If the Quick Entry Bar is enabled, you can record common entries faster on the ledger page. It is useful for frequent small expenses or fixed categories.

You can turn this feature on or off through “Settings” → “Quick entry bar”. When turned off, the ledger page shows the normal add button.

3. Ledgers, Categories, and Currencies

Ledgers

A ledger manages an independent set of entries, categories, and statistics. You can create different ledgers for different purposes.

For example:

  • Daily ledger
  • Travel ledger
  • Reimbursement ledger
  • Pet ledger

The free version can create 3 ledgers. After upgrading to Pro, the number of ledgers is unlimited.

Ledger Settings

In Ledger Settings, you can manage options for the current ledger, including:

  • Ledger name
  • Default currency
  • Ledger color
  • Entry theme
  • Ledger background
  • Whether mood records are enabled
  • Face ID / Touch ID lock
  • Delete all location records

To open Ledger Settings, go to the ledger list, swipe left on the ledger, then tap “Edit”.

After changing ledger settings, tap the “Save” button in the top-right corner to save them.
If you tap “Cancel” or leave the page directly, unsaved changes will be reverted.

Some settings require Pro. For example, ledger background is a Pro-only feature, and some entry themes also require Pro.

If you turn off mood records, the mood option will no longer appear when adding or editing entries. Existing mood data will not be deleted automatically.

“Delete All Location Records” only removes location information from entries in the current ledger. It does not delete the entries themselves.

Categories

Categories are used to distinguish different types of income and expenses.

Common expense categories:

  • Food
  • Transportation
  • Shopping
  • Entertainment
  • Medical
  • Pets
  • Travel

Common income categories:

  • Salary
  • Bonus
  • Part-time work
  • Investment
  • Gift money

You can adjust categories based on your own habits.

Category Management

You can open Category Management through “Top-right Menu” → “Category Management”.

On the Category Management page, tap “+” in the top-right corner to add a category.

Edit a Category

On the Category Management page, tap the category you want to modify.

You can adjust the category name, icon, background color, and other details. Some customization options may require Pro.

Default categories cannot be edited.

Show or Hide Categories

Categories can be shown or hidden.

If you do not want to use a category for now, you can hide it. Hidden categories will not appear when selecting a category while adding or editing entries, but existing entries using that category will not be deleted.

You can show the category again later in Category Management.

Category Icons

Categories can have icons and background colors, making them easier to identify in the entry list and statistics.

Basic icons and background colors can be set on the category editing page. Go to “Top-right Menu” → “Category Management”, then tap the category you want to edit to modify its appearance.

If you upgrade to Pro, you can also set custom images for categories.

Multiple Currencies

Each ledger has a default currency. Statistics and budgets are calculated based on the default currency.

If you record entries in another currency, Entryo converts them to the ledger’s default currency using the corresponding exchange rate before including them in statistics and budgets.

If an exchange rate has not been set, you may need to manually enter or confirm it when recording a non-default currency entry.

Exchange Rates

Exchange rates are used to convert non-default currencies to the ledger’s default currency.

You can enter exchange rates in two ways:

  1. When adding or editing an entry, select a non-default currency and enter the exchange rate used by that entry.
  2. Go to “Top-right Menu” → “Exchange Rates, then tap “+” in the top-right corner to add an exchange rate.

On the Exchange Rate Management page, you can tap an existing exchange rate to edit it, or swipe left on an exchange rate to delete it.

Note: Editing an exchange rate in Exchange Rate Management does not affect entries that have already been created. Existing entries keep the exchange rate used at the time they were recorded.

To change the exchange rate for a specific entry, go to the entry list, swipe right on the entry, tap “Edit”, then modify the exchange rate for that entry.

Changing the exchange rate for one entry only affects that entry. It does not affect other exchange rates in Exchange Rate Management.

4. Statistics and Budgets

Statistics Page

The statistics page is used to view income, expenses, and trends.

On the ledger page, tap the pie chart icon in the top-right corner to open the statistics page.

The statistics page includes:

ContentDescription
Income statisticsView income amount, trends, and categories
Expense statisticsView expense amount, trends, and categories
Category shareView how different categories contribute to income or expenses
Trend chartView income, expense, or mixed trend changes
Income goalSet a monthly income goal and track achievement progress
Net amountView income minus expenses
Previous month comparisonCompare the current month with the previous month
Category detailsView entries and statistics for a category

You can view data for different periods based on the selected time range.

Time Range

Statistics can be viewed by different time ranges, such as:

  • Today
  • This month
  • This year
  • Custom time range
  • All data

Different charts may support different time ranges.

Category Statistics

Category statistics help you understand where most of your money goes, or where your income comes from.

On the statistics page, tap a category to open the category details page and view detailed data and entries for that category.

The category details page usually includes:

  • Total income or expense for the selected time range
  • Entry list for the category
  • Custom time range filter
  • Same-period income or expense comparison
  • Trend chart
  • Category budget progress, if a category budget has been set

You can use a custom time range to view how a category changed during a specific period. For example, you can check food expenses during a trip, or shopping expenses over the last three months.

If a category budget has been set, the category details page also shows budget usage to help you see whether spending is approaching or exceeding the budget.

Trend Chart

The trend chart shows changes in income or expenses. On the statistics page, swipe left on the pie chart page to switch to the trend chart.

You can use the trend chart to observe:

  • Which month had higher expenses
  • Whether income is stable
  • Whether spending is concentrated in certain periods
  • How long-term habits change over time

Income Goal Supported in version 1.0.1 and later

You can set a monthly income goal to track whether your income reaches the expected amount.

On the Statistics page, switch to Income and select Month. The Income Goal card appears in the ledger’s default currency. Tap Goal Settings, enter the goal amount, and save.

In the monthly view, Entryo shows current income, the remaining amount or amount over goal, and the achieved percentage. After a goal is set, the yearly view can also show the monthly income goal trend. In this trend, each point shows that month’s difference from the goal: points above the dashed line mean over goal, points below the dashed line mean short of goal, and gray points mean there were no income entries that month. The dashed line means income equals the monthly goal.

Income goals are calculated in the ledger’s default currency. Income in other currencies is converted using the configured exchange rate and counted toward the goal. The Income Goal card is shown when viewing statistics in the ledger’s default currency.

Monthly Budget

You can set a monthly budget to control monthly spending.

Budgets are calculated using the ledger’s default currency. Entries in other currencies are converted using exchange rates and counted toward the budget.

Budget Available Amounts

If a monthly budget is set, Entryo may show the amount remaining for today, the amount available per remaining day, or the amount by which the monthly budget has been exceeded.

Remaining Today

When spending this month is below the safe budget calculated by time progress, Entryo shows the amount remaining for today.

This value helps you understand how much budget is still available today based on the current monthly progress.

In general:

Remaining today = safe budget based on time progress - spending this month

Available per Remaining Day

When spending this month has reached or exceeded the safe budget calculated by time progress, but has not exceeded the monthly budget, Entryo shows the amount available per remaining day.

This value helps you understand how much can be used each day if the remaining budget is spread across the remaining days of the month.

In general:

Available per remaining day = remaining budget ÷ remaining days of the month

Budget Exceeded

If spending this month exceeds the monthly budget, Entryo shows the amount by which the monthly budget has been exceeded.

Category Budgets

Pro users can set budgets for categories. Category budgets are useful for more detailed spending control, such as:

  • Food budget
  • Shopping budget
  • Entertainment budget
  • Pet budget

You can go to “Statistics Page” → “Budget Settings” → “Open Budget Allocation” to set category budgets.

Note: “Open Budget Allocation” is only visible to Pro users. If you have not upgraded to Pro, you can only set a monthly budget and cannot allocate budgets to individual categories.

Category budgets help you identify which specific categories are exceeding expectations. After setting them, the category details page will also show budget usage for the corresponding category.

5. Mood

What Are Mood Records?

Mood records let you record how you felt when an entry occurred. They can help you review the relationship between spending and emotions.

Examples:

  • Happy
  • Worth it
  • Impulsive
  • Regretful
  • Anxious
  • Calm
  • Looking forward
  • Relaxed

Mood is optional. You can record it only when needed.

When to Record Moods

Mood records are useful when you want to:

  • Understand which emotions make you more likely to spend
  • Distinguish “worth it” spending from “regretful” spending
  • Review your spending state during a certain period
  • Record expenses with feelings, such as travel, gifts, or entertainment

Mood Statistics

If the current ledger has mood records, the statistics page will show a mood card. Tap “View Mood Report” on the mood card to open the mood report page.

Pro users can view complete mood statistics. Mood statistics can help you understand:

  • Which moods appear more frequently
  • Which moods are associated with higher spending
  • Which purchases are more likely to lead to regret or satisfaction

If there are no mood records, the statistics page may not show the mood card. You can add mood records when adding or editing entries.

Mood Icons

You can manage mood labels and icons through “Top-right Menu” → “Mood Tags”.

On the Mood Labels page, you can pair mood icons with labels based on your own preferences.

Basic mood icons can be used directly. If you upgrade to Pro, you can also use your own images as mood icons.

6. Location

What Are Location Records?

Location records save where an entry occurred. For example:

  • Restaurant
  • Coffee shop
  • Shopping mall
  • Supermarket
  • Travel location

Location is optional. You can record it only when needed.

When to Record Locations

Location records are useful when you want to:

  • Know where you spend the most
  • Review travel expenses
  • Record frequently visited merchants
  • Analyze spending habits across different areas

Location Records and Location Permission

Entryo requests location permission only when you actively use your current location or enable entry location recording.

You can turn location recording on or off through “Settings” → “Record Entry Location”. When turned off, new locations will no longer be recorded when adding or editing entries, but existing location data will not be affected.

You can also turn off “Show place in entries”. When turned off, location information will not be shown on the entry page or category details page even if entries already have location data.

To delete existing location data, open Ledger Settings: in the ledger list, swipe left on the ledger, tap “Edit”, then tap “Delete All Location Records” at the bottom of the page.

Note: “Delete All Location Records” only removes location information from entries in the current ledger. It does not delete the entries themselves.

Location information is only used to save locations to your entries. It is not used to track you.
For details about location data and privacy, see the Privacy Policy.

Location Statistics

If the current ledger has location records, the statistics page will show a location card. Tap “View places and ranking” on the location card to open the location statistics page.

Location records can be used for free, but complete location statistics require Pro.

Location statistics help you understand spending from a spatial perspective and usually include:

  • Map view
  • Location spending ranking
  • Entries related to locations
  • Income and expense switching
  • Amount and record count switching

On the location statistics page, the map and ranking can work together. Tap a map marker or ranking item to view entries related to that location.

If there are no location records, the statistics page may not show the location card. You can add location records when adding or editing entries.

Delete Location Records

If you no longer want to keep location information, you can remove the location from a single entry or delete all location records in the current ledger.

To remove the location from one entry, swipe right on the entry in the entry list, tap “Edit”, then choose “Remove Location” on the entry editing page.

After removing the location, the entry itself remains, but its location information is deleted.

To delete all location records in the current ledger, open Ledger Settings: in the ledger list, swipe left on the ledger, tap “Edit”, then tap “Delete All Location Records” at the bottom of the page.

Note: “Delete All Location Records” only removes location information from entries in the current ledger. It does not delete the entries themselves.

7. Import and Export

Entryo supports Full Backup and Spreadsheet Import/Export. They are used for different purposes.

Full Backup

Full Backup is suitable for ledger migration or restore.

In the ledger list, swipe right on the ledger, tap “Export”, then choose “Full Backup”.

A Full Backup preserves as much complete ledger data as possible, such as:

  • Entries
  • Categories
  • Ledger settings
  • Images
  • Background
  • Icons
  • Subscriptions
  • Other settings data

Use Full Backup if you want to back up an entire ledger.

Restore from Backup

If you previously exported a ledger backup with “Full Backup”, you can import it again through “Restore from Backup”.

Go to “Top-right Menu” → “Import” → “Restore from Backup” to select a backup file.

Restore from Backup is suitable for:

  • Restoring a previously exported ledger when you are not using iCloud sync
  • Moving a ledger to another Apple ID or a device that does not use iCloud
  • Restoring from a Full Backup when iCloud data is unavailable
  • Restoring a ledger copy from a historical backup version

If iCloud sync is enabled on the current device and your previous ledger data has already been synced to iCloud, Entryo may automatically restore ledgers from iCloud.
In this case, you usually do not need to use “Restore from Backup”.

If you import the same ledger backup after iCloud has already restored it, the ledger list may show two ledgers with the same content. This does not affect the original data. You can keep the one you need and delete the extra ledger.

Restore from Backup preserves as much complete ledger data from the backup file as possible, such as:

  • Entries
  • Categories
  • Ledger settings
  • Images
  • Ledger background
  • Category icons
  • Entry themes
  • Other app-specific data

Note: Restore from Backup requires a Full Backup file exported by Entryo. Spreadsheet files cannot be used for full restore.

If you only want to import entries in bulk, use “Spreadsheet Import”.
If you need to restore a complete ledger, use a backup file exported with “Full Backup” and import it through “Restore from Backup”.

Spreadsheet Export

Spreadsheet Export is suitable for viewing, organizing, analyzing, or sharing entry data.

In the ledger list, swipe right on the ledger, tap “Export”, then choose “Spreadsheet Export”.

The exported CSV file can be opened in:

  • Excel
  • Numbers
  • Google Sheets
  • Other tools that support CSV

Note: Spreadsheet Export is not a Full Backup.

Spreadsheets usually only include entry fields, such as:

  • Type
  • Amount
  • Currency
  • Category
  • Note
  • Time
  • Exchange rate
  • Mood
  • Location
  • Whether excluded from statistics

Spreadsheets do not include complete data such as images, themes, ledger settings, or category icons.

Spreadsheet Import

You can use a CSV template to import entries in bulk.

Go to “Top-right Menu” → “Import” → “Spreadsheet Import” to select a CSV file.

Spreadsheet Import is useful for:

  • Migrating entry records from other accounting tools
  • Importing after organizing data in Excel or Numbers
  • Adding historical entries in bulk
  • Quickly importing many records

Note: Spreadsheet Import only creates entry records based on the spreadsheet content. It does not restore complete ledger data like “Restore from Backup”.
Images, ledger settings, ledger background, entry themes, category icons, and similar data are not restored through Spreadsheet Import.

If you need to migrate or restore a complete ledger, use a backup file exported with “Full Backup” and import it through “Restore from Backup”.

Spreadsheet Template Format

Spreadsheet Import currently only supports CSV files. Use the spreadsheet template provided by Entryo and do not modify the headers arbitrarily.

Go to “Top-right Menu” → “Import” → “Spreadsheet Import”, then tap “Download Spreadsheet Template” at the bottom of the page to get the template file.

After downloading the template, fill in entry data according to the field descriptions in the template. The template indicates which fields are required and which are optional.

After filling it in, return to the Spreadsheet Import page and select the CSV file to import it.

Amount and Type

The spreadsheet template includes “Type” and “Amount” fields.

“Type” is used to distinguish income from expense. Follow the template instructions and enter the income or expense text in the corresponding language, for example:

  • English template: Income / Expense

Enter positive numbers for “Amount”. Do not enter negative numbers.
Income or expense is determined by the “Type” field, not by plus or minus signs.

Date Format

The recommended fixed date format is:

yyyy-MM-dd HH:mm:ss

For example:

2026-06-08 14:30:00

Import may support several common date formats, but to avoid parsing errors, use the format shown in the template whenever possible.

8. iCloud Sync

What Is iCloud Sync?

If you enable iCloud, devices under the same iCloud account can sync ledger data. In general, these devices need to be signed in with the same Apple ID.

For example, you can record an entry on your iPhone and view it on an iPad signed in with the same Apple ID.

Sync May Take Time

ICloud sync is not instant upload and download. After installing the app on a new device, switching devices, using a slow network, or when iCloud status changes, ledgers may not appear completely right away.

If you previously used Entryo with iCloud sync enabled, but do not see data on a new device yet, wait for a while first. During sync, ledgers may appear gradually, or some ledgers may appear before others.

You can also reopen the app later to check the sync result. Sync time depends on network status, iCloud status, and device state.

Empty Ledger Page

If you open the app and see “No ledgers yet”, there are two possible cases:

  1. You are a new user and have not created a ledger yet.
  2. You previously used iCloud sync and data is still syncing.

You can wait for a while, then check again after confirming your network and iCloud status.

Before Using iCloud Sync

Please confirm:

  • The current device is signed in to iCloud
  • iCloud Drive is available
  • Entryo is allowed to use iCloud
  • Network connection is normal
  • Multiple devices use the same Apple ID

Recently Deleted Does Not Sync

Recently Deleted only keeps records on the current device. It does not sync via iCloud and is not exported with ledger backups.

This means:

  • Entries deleted on device A may not appear in Recently Deleted on device B
  • Recently Deleted is mainly for recovering accidental deletions on the current device
  • Full Backup does not include records in Recently Deleted

If you need to keep an accidentally deleted entry, restore it first, then create a Full Backup.

When Sync Has Not Updated for a Long Time

If you have waited for a while but ledgers still do not appear, or only some ledgers appear, check the following:

  • Whether the current device is connected to the network
  • Whether you are signed in with the correct Apple ID
  • Whether iCloud Drive is enabled
  • Whether Entryo is allowed to use iCloud
  • Whether the device is in low power, weak network, or a system state that limits syncing

You can also reopen the app later, or restart the device and check again.

If you urgently need to use a ledger on the current device, you can export a “Full Backup” of that ledger from another device that already has the data, then import it on the current device through “Restore from Backup”.

Note: If iCloud later syncs the same ledger back, the ledger list may show two ledgers with the same content. This does not affect the original data. You can keep the one you need and delete the extra ledger.

Avoid repeatedly uninstalling and reinstalling the app. Uninstalling the app may clear local data, and syncing iCloud data back may also take time.

Pro unlocks the complete Entryo feature set. Pro is a one-time purchase, not a subscription.

Unlimited Capacity

  • Unlimited subscriptions

Personalized Appearance

Deeper Statistics

10. URL Scheme and Shortcuts Supported in version 1.0.1 and later

Entryo can open the add-entry page with the entryo:// URL scheme. You can use it from iOS Shortcuts, a QR code, or another app to open Entryo with entry information already filled in.

The URL opens the add-entry page and pre-fills available fields. You still need to review and save the entry in Entryo.

URL format

entryo://add-entry?book=Main%20Ledger&amount=12.5&note=Coffee&date=2026-06-23&currency=USD&exchangeRate=1&type=expense&category=Food&mood=Happy

If you build the URL manually or from variables, text values such as ledger, note, category, and mood can usually be entered as shown in Entryo. If the text contains spaces or special characters, URL encoding can help avoid recognition issues.

Supported parameters

ParameterValueDescription
bookLedger name, for example Main LedgerTarget ledger. If omitted, Entryo uses the current ledger.
amountNumber, for example 12.5Entry amount.
typeexpense or incomeEntry type. expense means expense, and income means income. Defaults to expense when omitted.
categoryCategory name as shown in Entryo, for example Food, Salary, Transport, ShoppingPreselects the category when the name exists in the target ledger. Custom category names can also be used.
noteText, for example CoffeeEntry note.
dateyyyy-MM-dd, yyyy-MM-dd HH:mm, or ISO 8601Entry date. Future dates are adjusted to the current time.
currencyCurrency code, for example USD, CNY, JPYIf omitted, Entryo uses the ledger default currency.
exchangeRateNumber, for example 1Exchange rate used to convert the entry currency into the ledger default currency.
moodEnabled mood label name as shown in Entryo, for example Happy, Neutral, Calm, RelaxedPreselects the mood when mood records are enabled for the ledger and the specified mood label is enabled.

If the specified ledger, category, mood, or currency cannot be found, Entryo will open the add-entry page with the remaining valid fields. URL Scheme will not create new ledgers, categories, currencies, or mood labels automatically. For moods, only mood labels enabled in the target ledger can be selected.

Example

entryo://add-entry?book=Main%20Ledger&amount=12.5&note=Coffee&date=2026-06-23&currency=USD&exchangeRate=1&type=expense&category=Food&mood=Happy

This example opens Entryo, selects “Main Ledger”, and opens the add-entry page with a USD 12.5 coffee expense.

Use with iOS Shortcuts

  1. Open the Shortcuts app and create a new shortcut.
  2. Add the URL action, then enter or build an entryo://add-entry?... URL.
  3. Add the Open URL action after it.
  4. Run the shortcut. iOS will open Entryo and show the pre-filled add-entry page.

11. FAQ

Q1: Why do I not see my previous ledgers after installing the app on a new device?

ICloud sync may not have finished yet, or there may be no synced data under the current Apple ID.

Please check:

  • Whether you are signed in with the original Apple ID
  • Whether iCloud Drive is enabled
  • Whether the network is normal
  • Whether Entryo is allowed to use iCloud

Sync time depends on network status, iCloud status, and device state. After installing on a new device or switching devices, ledgers may appear gradually, or some ledgers may appear before others.

For more details, see Sync May Take Time.

Q2: Can Spreadsheet Export be used as a Full Backup?

Not recommended.

Spreadsheet Export is mainly for viewing, organizing, and analyzing entry data. It does not include complete information such as images, themes, ledger settings, or category icons.

If you need to migrate or restore a complete ledger, use Full Backup.

Q3: Can exported spreadsheet files be imported again?

Yes. Spreadsheet files can be used to import entries again.

However, re-importing only restores the entry fields contained in the spreadsheet. It does not restore complete ledger settings, images, themes, or other app-specific data.

Q4: Why is the amount in statistics different from the amount I entered?

Possible reasons include:

  • You recorded entries in a non-default currency. Entryo converts them to the ledger’s default currency using exchange rates before including them in statistics.
  • Some entries have “Exclude from daily stats” enabled, so they are not included in statistical amounts by default.
  • The current time range, income/expense type, or category filter on the statistics page may differ, causing the displayed amount to differ from the amount in the entry list.

You can check the entry’s currency, exchange rate, and “Exclude from daily stats” setting.

If you need to view complete data including these entries, turn on “Show All Entries” on the statistics page.

Q5: Why does the budget amount not match the expense amount?

Possible reasons include:

  • Some entries use non-default currencies and are converted by exchange rates
  • Some entries are excluded from daily statistics
  • The current time range is different
  • Category filters affect part of the statistics display
  • Budgets are calculated in the ledger’s default currency

Please refer to the explanation shown on the budget page in the app.

Q6: Can I restore an entry after deleting it?

You can restore some deleted records in Recently Deleted.

Note: Recently Deleted only keeps records on the current device, and the number of records retained is limited.

If you need to keep data long-term, create Full Backups regularly.

Q7: Does Recently Deleted sync to other devices?

No.

Recently Deleted only keeps deleted records on the current device and does not sync via iCloud.

Q8: Does Entryo upload my accounting data?

Your accounting data is stored locally. If iCloud sync is enabled, data syncs between devices through your personal iCloud account.

Entryo does not read your iCloud data, and does not sell or share your personal data.

For details about data use, iCloud sync, location permission, and Face ID / Touch ID, see the Privacy Policy.

Q9: Why does the location feature need location permission?

Entryo requests location permission only when you actively use your current location, record entry locations, or use location-related features.

Location data is only used to save locations to corresponding entries. It is not used to track your location.

For details about location permission and location data use, see the Privacy Policy.

Q10: Is Pro a subscription?

No.

Pro is a one-time purchase, not a monthly or yearly subscription. After purchase, it unlocks the complete Entryo feature set, including unlimited ledgers, unlimited subscriptions, personalized appearance, mood statistics, location statistics, and category budgets.

Q11: Do I need to purchase Pro again after switching devices?

No.

As long as you use the same Apple ID used to purchase Pro, you can restore Pro through “Settings” → “Upgrade to Pro” → “Already purchased? Restore Purchase”.

For more details, see Restore Purchase.

Q12: Why can I not find some feature entrances?

Some features are shown depending on the current data state. For example:

  • If there is no location data, location-related entrances may not be obvious
  • If there is no mood data, mood statistics may have no content
  • Some Pro features require purchase before use
  • Some settings are shown inside a ledger, rather than in global settings

First check whether you are currently on the ledger page, statistics page, or settings page.

Q13: When should I use “Exclude from daily stats”?

Use it for entries that you do not want to affect daily spending analysis, such as:

  • Paying on behalf of someone else
  • Reimbursement
  • Temporary adjustments
  • Special expenses that are not part of daily spending

These entries remain in the list, but are not included in related statistics.

Q14: How do I contact support?

You can contact support through “Settings” → “Report an Issue”.

When reporting an issue, include:

  • Device model
  • iOS version
  • App version
  • Screenshots
  • Steps to reproduce
  • Error code, if the issue is purchase-related

Do not send bank card numbers, passwords, verification codes, or other sensitive information in feedback.
